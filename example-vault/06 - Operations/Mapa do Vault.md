Este mapa ficticio demonstra como Codex pode se orientar em um vault Obsidian.

## Areas principais

- `00 - Inbox/` - capturas ainda sem processamento.
- `01 - Source Materials/` - fontes brutas ou processadas.
- `02 - Knowledge/` - notas conceituais permanentes.
- `06 - Operations/` - logs, decisoes e mapas.

## Como usar

- Para transformar uma fonte em nota util, use a skill `processar-fonte`.
- Para criar uma nota permanente, use a skill `criar-conceito`.
- Para melhorar navegacao, use a skill `conectar-notas`.
- Para registrar escolhas importantes, use a skill `registrar-decisao`.
- Para registrar sessoes de trabalho, use a skill `registrar-log`.
