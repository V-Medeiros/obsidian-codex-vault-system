# Criar Conceito

## Objetivo

Criar notas de conhecimento reutilizaveis, conectadas e faceis de consultar depois.

## Workflow

1. Verificar se ja existe nota com o mesmo nome ou nome equivalente.
2. Se existir, propor edicao incremental em vez de duplicar.
3. Escolher subpasta em `02 - Knowledge/`:
   - `Tecnologia/`
   - `Leitura/`
   - `Pessoas/`
   - `Outros/`
4. Criar frontmatter `type: concept`.
5. Escrever explicacao objetiva e exemplos.
6. Adicionar conexoes para notas existentes ou alvos uteis.
7. Se a nota veio de uma fonte, linkar a fonte em `01 - Source Materials/`.

## Template

Ao criar ou reestruturar uma nota conceitual, leia `references/template-concept.md`.

## Regras

- Nao criar nota duplicada se uma nota equivalente ja existir.
- Manter titulo e filename iguais ou muito proximos.
- Preferir linguagem clara, concreta e consultavel.
- Evitar texto generico que nao ajude revisao futura.
- Usar poucos wikilinks de alta qualidade.
- Evitar notas conceituais que sejam apenas resumo de uma fonte.
- Se a nota ficar grande demais, extrair sub-conceitos para novas notas.
- Evitar definicoes no formato "X nao e..., e Y"; escrever primeiro a definicao direta e usar contrastes apenas quando forem necessarios.
