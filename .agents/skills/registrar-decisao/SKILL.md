# Registrar Decisao

## Objetivo

Salvar decisoes de forma rastreavel para que Codex e o usuario entendam depois por que algo foi escolhido.

## Destino

Registrar em `06 - Operations/Decisoes.md`, adicionando uma entrada nova no topo da lista ou na secao cronologica adequada.

## Workflow

1. Identificar a decisao real em uma frase.
2. Registrar contexto suficiente para uma leitura futura.
3. Listar alternativas consideradas, mesmo que brevemente.
4. Registrar consequencias e tradeoffs.
5. Definir proximos passos.
6. Linkar projeto, conceito ou fonte relacionada com `[[wikilinks]]`.

## Formato

```markdown
## YYYY-MM-DD - Titulo da decisao

### Contexto

### Decisao

### Alternativas Consideradas

### Consequencias

### Proximos Passos

### Links
- [[Projeto ou conceito]]
```

## Regras

- Registrar fatos e motivos, nao apenas conclusoes.
- Se a decisao ainda nao estiver tomada, registrar como proposta ou pendencia.
- Nao editar decisoes antigas para reescrever a historia; adicionar uma nova entrada se a decisao mudar.
